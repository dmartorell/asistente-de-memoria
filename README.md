# Asistente de Memoria

Sistema de gestión de mensajería.

## Descripción

Aplicación para la gestión y administración de servicios de mensajería.

## Instalación

```bash
npm install
```

## Uso

```bash
npm start
```

## Configuración

Copia el archivo `.env.example` a `.env` y configura las variables de entorno necesarias.
